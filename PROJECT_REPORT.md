## ⚡ 核心技术亮点

### 线程安全设计
```cpp
// Document 使用 shared_mutex 实现多读单写
std::shared_mutex mutex_;

// 读操作（并发安全）
{
    std::shared_lock lock(mutex_);
    std::string text = full_text_;
}

// 写操作（互斥安全）
{
    std::unique_lock lock(mutex_);
    full_text_ = new_text;
}
```

### 异步 UI 更新
```cpp
// 后台线程中调用 AI
QtConcurrent::run([this]() {
    auto result = ai_writer_->polish(text);
    // 安全地回到 GUI 线程
    QMetaObject::invokeMethod(this, [this, result]() {
        ui_->setText(result);
    }, Qt::QueuedConnection);
});
```

### 流式响应处理
```cpp
ai_client_->completeStream(prompt,
    [this](const StreamChunk& chunk) {
        if (!chunk.text.empty()) {
            // 逐字显示，实时反馈
            display_->append(chunk.text);
        }
        if (chunk.is_finish) {
            // 流式结束
            finalize();
        }
    });
```
